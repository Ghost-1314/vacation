# 向量

- 线性组合：**给几个向量，每个向量乘一个常数（叫 “标量”），然后把结果加起来，得到的新向量就是这几个向量的线性组合**。

```python
import numpy as np 

# 1. 定义两个基础向量（2维向量，方便理解） 
v1 = np.array([2, 3]) # 向量v1 = [2, 3] 
v2 = np.array([4, 1]) # 向量v2 = [4, 1] 

# 2. 定义标量（系数） 
a = 2 # 第一个系数 
b = -1 # 第二个系数 

# 3. 计算线性组合：
y = a*v1 + b*v2 

# 步骤拆解：先分别乘系数，再相加（NumPy支持逐元素运算） 
step1 = a * v1 # 2*[2,3] = [4,6] 
step2 = b * v2 # -1*[4,1] = [-4,-1] 
y = step1 + step2 # [4-4, 6-1] = [0,5] 
# 输出结果 
print("向量v1：", v1) 
print("向量v2：", v2) 
print(f"{a}×v1的结果：", step1) 
print(f"{b}×v2的结果：", step2) 
print("线性组合结果y = 2*v1 -1*v2：", y) 

# 也可以一步计算（更简洁） 
y_short = a * v1 + b * v2 print("一步计算的结果：", y_short)
```

```python
import numpy as np 

# 特征向量：[学历, 工作年限] 
feature1 = np.array([1, 2, 3]) # 学历（1=本科，2=硕士，3=博士） 
feature2 = np.array([2, 5, 8]) # 工作年限 

# 系数（权重）：学历权重3k，工作年限权重5k 
w1 = 3000 
w2 = 5000 

# 薪资 = w1×学历 + w2×工作年限（线性组合） 
salary = w1 * feature1 + w2 * feature2 
print("预测薪资：", salary) # 输出 [13000 31000 49000]
```

- 点积：向量的点积是**两个维度相同的向量**之间的一种运算，结果不是向量，而是一个**标量（单个数字）**。

```python
import numpy as np 

# 1. 定义两个维度相同的向量（必须同维度！） 
a = np.array([2, 3, 4]) # 3维向量 
b = np.array([5, 6, 7]) # 3维向量 

# 方式1：手动计算（理解底层逻辑） 
dot_manual = a[0]*b[0] + a[1]*b[1] + a[2]*b[2] 
print("手动计算点积：", dot_manual) # 2×5 + 3×6 + 4×7 = 10+18+28=56 

# 方式2：NumPy内置函数（推荐，简洁高效） 
dot_np1 = np.dot(a, b) 
print("np.dot计算点积：", dot_np1) # 结果56 

# 方式3：Python 3.5+的运算符（更简洁） 
dot_np2 = a @ b # 注意：@是矩阵/向量乘法运算符，一维数组时等价于点积 
print("@运算符计算点积：", dot_np2) # 结果56 

# 反例：维度不同的向量无法计算点积（会报错） 
c = np.array([1, 2]) # 2维向量 
# print(np.dot(a, c)) # 取消注释会报错：ValueError: shapes (3,) and (2,) not aligned
```

- **维度必须一致**：这是点积的前提，比如 3 维向量只能和 3 维向量算点积，否则 Python 会报错；
- **结果是标量**：和线性组合（结果是向量）最大的区别 —— 点积算完是一个数，不是数组；
- **`NumPy`的优势**：当向量维度很大（比如 1000 维），手动计算不现实，`np.dot()`是底层优化的，速度极快。


- 点积的实际应用：

在 Python 的实际开发中，点积的应用无处不在：

#### 1. 机器学习：计算相似度

比如推荐系统中，用用户的 “评分向量” 计算相似度：

- 用户 A 的评分向量：`[5, 3, 4]`（对电影 1、2、3 的评分）；
- 用户 B 的评分向量：`[4, 4, 5]`；
- 点积越大，说明两个用户的口味越像，可基于此推荐电影。

#### 2. 线性代数：计算投影 / 判断垂直

- 点积为 0 → 两个向量垂直（比如 `[1,0]` 和 `[0,1]`）；
- 点积的结果除以其中一个向量的模长，就是另一个向量在这个向量上的投影长度。

#### 3. 神经网络：核心计算

神经网络中，“输入 × 权重” 的核心就是点积：

- 输入向量：`[x1, x2, x3]`；
- 权重向量：`[w1, w2, w3]`；
- 神经元的线性输出：`x1w1 + x2w2 + x3w3`（就是点积）。


# 矩阵

## 矩阵乘法

#### 1. 运算前提（维度匹配规则）

只有当**第一个矩阵的列数 = 第二个矩阵的行数**时，才能相乘。

- 记法：若矩阵A是 `m×n`（m行n列），矩阵B是 `n×p`（n行p列），则A×B的结果是 `m×p` 矩阵；
    
- 反例：3×2矩阵（3行2列）只能和2×4矩阵相乘，不能和3×4矩阵相乘（列数2≠行数3）。
    
---
#### 2. 计算规则（核心逻辑）

结果矩阵中第`i`行第`j`列的元素 = 矩阵A的第`i`行向量 · 矩阵B的第`j`列向量（也就是这两个向量的点积）。

**通俗拆解（以2×2矩阵为例）**：

假设：

$$A = \begin{bmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{bmatrix}, \quad B = \begin{bmatrix} b_{11} & b_{12} \\ b_{21} & b_{22} \end{bmatrix}$$

则：

$$A \times B = \begin{bmatrix} (a_{11}×b_{11} + a_{12}×b_{21}) & (a_{11}×b_{12} + a_{12}×b_{22}) \\ (a_{21}×b_{11} + a_{22}×b_{21}) & (a_{21}×b_{12} + a_{22}×b_{22}) \end{bmatrix}$$

简单说：**行乘列，对应相乘再求和**（本质就是向量点积）。

---
#### 3.编程实现

```Python
import numpy as np

# 1. 定义两个符合维度规则的矩阵
# A：2×3矩阵（2行3列），B：3×2矩阵（3行2列）→ 结果应该是2×2矩阵
A = np.array([
    [1, 2, 3],  # A的第1行
    [4, 5, 6]   # A的第2行
])
B = np.array([
    [7, 8],     # B的第1列
    [9, 10],    # B的第2列
    [11, 12]    # B的第3列
])

# 方式1：手动计算（理解底层逻辑，仅用于学习）
# 结果矩阵C是2×2，逐个计算每个元素（行×列的点积）
C = np.zeros((2, 2))  # 初始化2×2的零矩阵
# C[0,0] = A第1行 · B第1列 = 1×7 + 2×9 + 3×11 = 7+18+33=58
C[0, 0] = A[0,0]*B[0,0] + A[0,1]*B[1,0] + A[0,2]*B[2,0]
# C[0,1] = A第1行 · B第2列 = 1×8 + 2×10 + 3×12 = 8+20+36=64
C[0, 1] = A[0,0]*B[0,1] + A[0,1]*B[1,1] + A[0,2]*B[2,1]
# C[1,0] = A第2行 · B第1列 = 4×7 + 5×9 + 6×11 = 28+45+66=139
C[1, 0] = A[1,0]*B[0,0] + A[1,1]*B[1,0] + A[1,2]*B[2,0]
# C[1,1] = A第2行 · B第2列 = 4×8 + 5×10 + 6×12 = 32+50+72=154
C[1, 1] = A[1,0]*B[0,1] + A[1,1]*B[1,1] + A[1,2]*B[2,1]
print("手动计算矩阵乘法结果：\n", C)

# 方式2：NumPy内置函数（推荐，高效）
C_np1 = np.matmul(A, B)  # matmul = matrix multiplication（矩阵乘法）
print("\nnp.matmul计算结果：\n", C_np1)

# 方式3：@运算符（Python 3.5+，最简洁）
C_np2 = A @ B  # 一维数组=点积，二维数组=矩阵乘法
print("\n@运算符计算结果：\n", C_np2)

# 反例：维度不匹配会报错
# B_error = np.array([[7,8],[9,10]])  # 2×2矩阵
# print(np.matmul(A, B_error))  # 取消注释报错：shapes (2,3) and (2,2) not aligned
```

- **维度匹配是前提**：一定要先检查“前矩阵列数=后矩阵行数”，比如上面A是2×3，B是3×2，列数3=行数3，才能相乘；
    
- **结果维度规律**：m×n 矩阵 × n×p 矩阵 = m×p 矩阵（上面2×3 × 3×2 = 2×2）；
    
- **和元素级乘法的区别**：
    
    - 矩阵乘法：`A @ B` → 按“行乘列点积”计算；
        
    - 元素级乘法：`A * B` → 仅当A和B维度完全相同（如都是2×2）时，对应位置元素相乘（新手极易混淆！）。

#### 4.实际应用
##### 1. 机器学习：特征变换

比如你有100个样本（每行一个样本），每个样本有5个特征（5列），即100×5的特征矩阵；

用一个5×8的权重矩阵去乘，就能把5维特征转换成8维特征（100×8）——这是降维/升维的基础（如PCA、神经网络层变换）。

##### 2. 图形学：坐标变换

比如用旋转矩阵（2×2）乘坐标矩阵（2×n），就能让n个坐标点整体旋转——OpenCV中图像旋转、缩放的底层就是矩阵乘法。

##### 3. 神经网络：核心计算

神经网络中“输入层→隐藏层”的计算：

- 输入矩阵（样本数×输入维度） × 权重矩阵（输入维度×隐藏层维度） = 隐藏层输出矩阵；
    
- 这是神经网络最核心的运算，没有矩阵乘法就没有深度学习。

## 矩阵转置
### 1. 转置的直观理解

矩阵转置就是把矩阵的**行和列互换**：原矩阵的第`i`行第`j`列元素，会变成转置矩阵的第`j`行第`i`列元素。

- 记法：矩阵`A`的转置记作`A^T`（上标T）；
    
- 维度变化：若原矩阵是`m×n`（m行n列），转置后就是`n×m`（n行m列）。
    

**举个简单例子（2×3矩阵转置）**：

原矩阵A（2行3列）：

$$A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix}$$

转置矩阵$$A^T$$（3行2列）：

$$A^T = \begin{bmatrix} 1 & 4 \\ 2 & 5 \\ 3 & 6 \end{bmatrix}$$

**特殊情况**：

- 行向量（1×n）转置后变成列向量（n×1）：比如`[1,2,3]^T = [[1],[2],[3]]`；
    
- 列向量（n×1）转置后变成行向量（1×n）；
    
- 方阵（n×n）转置后维度不变（仍是n×n），若转置后和原矩阵相等（$$A^T=A$$），则称为**对称矩阵**（比如单位矩阵、对角矩阵）。

---
### 2.代码实现
```Python
import numpy as np

# 1. 定义原始矩阵（2行3列）
A = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
print("原始矩阵A（2×3）：\n", A)
print("A的维度：", A.shape)  # 输出 (2, 3)

# 方式1：使用.T属性（最简洁，推荐）
A_T1 = A.T
print("\nA.T 转置结果（3×2）：\n", A_T1)
print("A.T的维度：", A_T1.shape)  # 输出 (3, 2)

# 方式2：使用np.transpose()函数（通用，支持高维张量）
A_T2 = np.transpose(A)
print("\nnp.transpose(A) 转置结果：\n", A_T2)

# 方式3：手动转置（理解底层逻辑，仅用于学习）
# 原矩阵是2×3 → 转置后是3×2，先初始化零矩阵
A_T3 = np.zeros((3, 2))
for i in range(A.shape[0]):  # 遍历原矩阵的行
    for j in range(A.shape[1]):  # 遍历原矩阵的列
        A_T3[j, i] = A[i, j]  # 行和列互换
print("\n手动转置结果：\n", A_T3)

# 特殊例子：对称矩阵（转置后和原矩阵相等）
B = np.array([
    [1, 2, 3],
    [2, 4, 5],
    [3, 5, 6]
])
print("\n对称矩阵B：\n", B)
print("B.T == B？", np.all(B.T == B))  # 输出 True
```

- **.T属性**：`NumPy`数组的内置属性，仅对二维数组转置时，等价于`np.transpose()`，是最常用的方式。

---
### 3.核心性质
转置有几个非常重要的运算性质，是矩阵操作的基础，我们结合代码验证：

#### 性质1：转置的转置等于原矩阵

$$(A^T)^T = A$$

```Python
# 验证：转置两次回到原矩阵
print("(A.T).T == A？", np.all((A.T).T == A))  # 输出 True
```


#### 性质2：和/差的转置 = 转置的和/差

$$(A + B)^T = A^T + B^T$$$$(A - B)^T = A^T - B^T$$

```Python
# 定义两个同维度矩阵
A = np.array([[1,2],[3,4]])
B = np.array([[5,6],[7,8]])
# 验证和的转置
left = (A + B).T
right = A.T + B.T
print("(A+B).T == A.T+B.T？", np.all(left == right))  # 输出 True
```


#### 性质3：数乘的转置 = 数×转置

$$(kA)^T = kA^T$$（**k是常数**）

```Python
k = 2
left = (k * A).T
right = k * A.T
print("(k*A).T == k*A.T？", np.all(left == right))  # 输出 True
```


#### 性质4：乘积的转置 = 转置的乘积（顺序互换！）

$$(AB)^T = B^T A^T$$（这是最常用也最容易错的性质）

```Python
# 定义符合乘法规则的矩阵：A(2×3), B(3×2)
A = np.array([[1,2,3],[4,5,6]])
B = np.array([[7,8],[9,10],[11,12]])
# 验证乘积的转置
left = (A @ B).T  # (AB)^T
right = B.T @ A.T  # B^T A^T（注意顺序！）
print("(A@B).T == B.T@A.T？", np.all(left == right))  # 输出 True
```

#### 关键解释：

- **乘积转置的顺序互换**：最容易犯的错是写成`A^T B^T`，必须记住“先转置，再倒序相乘”；
    
- **应用场景**：当需要计算`(AB)^T`但`AB`维度太大时，可通过`B^T A^T`优化计算速度（`NumPy`底层对转置和乘法的优化更高效）。

---
### 4.实际应用

#### 1. 解决矩阵乘法的维度匹配问题

比如有矩阵A（2×3）和矩阵C（2×3），想计算`A × C`但维度不匹配（A的列数3≠C的行数2），此时可转置C为3×2，满足`A(2×3) × C^T(3×2)`的乘法规则：

```Python
A = np.array([[1,2,3],[4,5,6]])
C = np.array([[7,8,9],[10,11,12]])
# A @ C 会报错（维度不匹配），但A @ C.T 可行
result = A @ C.T
print("A × C^T 的结果：\n", result)
```



#### 2. 机器学习：特征矩阵的维度调整

比如把“样本数×特征数”的矩阵（100×5）转置为“特征数×样本数”（5×100），方便计算特征的均值、方差等统计量；



#### 3. 线性代数：对称矩阵相关计算

比如求解线性方程组、矩阵求逆时，对称矩阵（$$A^T=$$）有更简洁的计算方法，转置是判断和构造对称矩阵的基础。

### 总结

1. **矩阵转置的核心**：行和列互换，维度从`m×n`变为`n×m`，`NumPy`中用`A.T`即可实现；
    
2. **核心性质**：转置的转置等于原矩阵、乘积的转置需倒序相乘（$$(AB)^T = B^T A^T$$）是最关键的两个性质；
    
3. **实际价值**：解决矩阵乘法维度匹配问题，是机器学习、线性代数计算中调整矩阵维度的核心操作。

## 矩阵的逆
### 一、什么是矩阵的逆？

- 只有**方阵**（n×n）才**有可能**有逆。
- 定义：对于 n 阶方阵 A，如果存在一个方阵 B，使得：AB=BA=I（I是单位矩阵）

那么：

- B 就是 A 的**逆矩阵**，记作 $$A^{-1}$$
- 此时称 A 是**可逆矩阵 / 非奇异矩阵**

一句话：**逆矩阵就是和原矩阵相乘等于单位矩阵的那个矩阵。**

**注意**：**单位矩阵**：主对角线（左上到右下）全1，其它都是0

### 三、逆矩阵的核心性质
设 A、B 都是 n 阶可逆方阵，k≠0 是常数：

1. $$(A^{-1})^{-1} = A $$ 逆的逆是自己。



2. $$(A^T)^{-1} = (A^{-1})^T$$转置和逆可交换顺序。



3. $$(kA)^{-1} = \frac{1}{k}A^{-1}$$数乘的逆 = 倒数 × 逆。



4. $$(AB)^{-1} = B^{-1}A^{-1}$$**乘积的逆 = 逆的倒序乘积**（和转置性质一模一样！）



5. $$AA^{-1} = A^{-1}A =I $$最基本定义。



6. 若 A 可逆，则 $$A^{-1}$$唯一。

---
### 四、什么时候没有逆
满足下面任意一条，A **不可逆**（奇异矩阵）：

- `det(A)` = 0
    
- 行/列线性相关
    
- 矩阵不满秩
    
- 方程组 Ax=0 有非零解

---
### 五、Python求逆

`numpy`：
```Python
import numpy as np

A = np.array([[1, 2],
              [3, 4]])

# 求逆
A_inv = np.linalg.inv(A)

print(A_inv)

# 验证 A * A_inv = I
print(np.round(A @ A_inv))
```

输出：

```Plain
[[-2.   1. ]
 [ 1.5 -0.5]]

[[1 0]
 [0 1]]
```

---
### 六、逆矩阵用来干嘛

- 解线性方程组：$$Ax=b$$


- 如果 A 可逆：$$x=A^{−1}b$$
这就是**矩阵版除法**。

---
### 七、总结
1. 只有**方阵**可能有逆
    
2. 可逆 ⇔ `det`≠0
    
3. 逆满足：$$(AB)^{-1}=B^{-1}A^{-1}$$
4. 求逆用 `np.linalg.inv(A)`
    
5. 作用：解方程组、做“矩阵除法”

## 行列式

### 一、定义
- 只有 **n×n** 方阵才有行列式，记作：$$det(A)$$或$$|A|$$
- 总结：行列式是一个**数**，用来衡量这个方阵**有没有逆、是否奇异、是否线性相关**。

### 三、行列式的核心意义

设 A 是 n 阶方阵：

1. **`det(A`) ≠ 0**
    ⇒ A **可逆**（非奇异）
    ⇒ 行/列线性无关
    ⇒ 方程组 Ax=b 有唯一解
    
2. **`det(A)` = 0**
    ⇒ A **不可逆**（奇异）
    ⇒ 行/列线性相关
    ⇒ 方程组 Ax=b 无解 or 无穷多解

---
### 四、行列式的核心性质
设 A、B 为 n 阶方阵：

1. **单位矩阵行列式 = 1**：$$\det(I) = 1$$
2. **交换两行/两列，行列式变号**
    
3. **某一行/列全为 0 ⇒ `det`=0**
    
4. **两行/两列成比例或相同 ⇒ `det`=0**
    
5. **某一行/列提公因子 k ⇒ `det` 也提 k**：$$\det(kA) = k^n \det(A$$
6. **某一行是两行之和，行列式可拆分**
    
7. **某一行加上另一行的倍数，行列式不变**（这是化简行列式最常用）
    
8. **转置不改变行列式**：$$\det(A^T) = \det(A$$
9. **乘积的行列式 = 行列式的乘积**：$$\det(AB) = \det(A)\det(B$$
10. **逆矩阵的行列式 = 倒数**：$$\det(A^{-1}) = \dfrac{1}{\det(A)}$$

---
### 五、Python计算

```Python
import numpy as np

A = np.array([[1,2],
              [3,4]])

det = np.linalg.det(A)
print(det)
```

输出：

```Plain
-2.0
```

### 六、总结

- 行列式是**方阵**才有的一个**数**
    
- `det`≠0 ⇔ 可逆
    
- `det`=0 ⇔ 不可逆
    
- 记住：`det(AB)=detA detB`、`det(A⁻¹)=1/detA`
    
- 计算：2×2 直接套公式，高阶用 Python

# 多元微分学

## 1.偏导

- 定义：**偏导 = 按住其他变量不动，只对一个变量求导。**

设函数 $$z = f(x,y)$$
则，对`x`求偏导，即是把`y`当做常数，只对`x`求导：$$\frac{\partial z}{\partial x} = f_x(x,y)$$
同理，对`y`求偏导：$$\frac{\partial z}{\partial y} = f_y(x,y)$$

- 示例：若$$z = x^2 y + 3y^3$$
则$$f_x = 2xy,\quad f_y = x^2 + 9y^2$$

---
## 2.全微分

### （1）定义

如果$$z=f(x,y)$$可微，则$$dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy$$
- 理解：
	- `dx`，`dy`是`x`，`y`的微小变化。
	- `dz`是函数值线性近似的变化。





### （2）可微的关系（重要）

$$\text{可微} \Rightarrow \text{偏导存在} \Rightarrow \text{连续}$$
反过来不一定成立。

---
## 3.方向导数&梯度（最实用）

### （1）方向导数

- 定义：函数**沿某一方向**的变化率。
- 设方向单位向量 $$\mathbf{u}=(\cos\alpha,\cos\beta)$$
则：$$\frac{\partial f}{\partial \mathbf{u}}

= f_x \cos\alpha + f_y \cos\beta$$



### （2）梯度（最重要）

$$\nabla f = \left( \frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y} \right)$$


- **梯度三句金律**：

	1. **梯度方向 = 函数上升最快的方向**
	    
	2. **负梯度方向 = 下降最快方向**（机器学习核心）
	    
	3. **梯度垂直于等值线/等值面**

- 方向导数可以用梯度简写：$$\frac{\partial f}{\partial \mathbf{u}} = \nabla f \cdot \mathbf{u}$$

---
## 4.多元函数极值

设$$z=f(x,y)\quad,在(x_0,y_0)处：$$
- 第一步，找驻点：$$f_x=0,\quad f_y=0$$


- 第二步，算三个数：$$A = f_{xx},\quad B = f_{xy},\quad C = f_{yy}$$
$$\Delta = AC - B^2$$


- 第三步，判断：
1. $$\Delta > 0,\ A>0\quad → 极小值$$
2. $$\Delta > 0,\ A<0\quad→ 极大值$$
3. $$\Delta < 0\quad → 鞍点（不是极值）$$
4. $$\Delta = 0 \quad→ 判别失效$$

---
## 5.条件极值（拉格朗日乘数法）

- 目标：$$在约束g(x,y)=0下，求f(x,y)的极值$$
即：**条件极值 = 把约束“吸进”函数里，变成无约束求导。**

- 万能步骤：
1. 构造拉格朗日函数：$$L = f(x,y) - \lambda g(x,y)$$
2. 对 x,y,λ 分别求偏导并令为 0：$$L_x=0,\quad L_y=0,\quad g(x,y)=0$$
3. 解方程组，得到可能极值点。


---
## 总复习（简）

1. **偏导**：只动一个变量，其他当常数
    
2. **全微分**：$$dz = f_x dx + f_y dy$$
    
3. **梯度**：$$\nabla f=(f_x,f_y)，上升最快方向$$
4. **多元极值**：$$求驻点 → 算 AC-B^2\quad判断$$
5. **条件极值**：拉格朗日乘数法

## 多元微分学在Python的应用

### 一、偏导数：Python数值求导的基础

偏导数是“对单个变量求导”，对应Python中**数值微分**（不用手推公式，直接用代码算导数），是所有优化算法的第一步。

#### 应用场景

- 机器学习中手动实现梯度下降（不用框架，自己写核心逻辑）
    
- 科学计算中分析单变量对结果的影响（比如分析温度对化学反应速率的影响）
    

#### Python代码示例（数值法求偏导）

```Python
import numpy as np

# 定义多元函数 z = x²y + 3y³
def f(x, y):
    return x**2 * y + 3 * y**3

# 数值法求偏导（对x）：固定y，用差分近似
def partial_derivative_x(f, x, y, h=1e-6):
    return (f(x + h, y) - f(x - h, y)) / (2 * h)  # 中心差分，精度更高

# 数值法求偏导（对y）：固定x
def partial_derivative_y(f, x, y, h=1e-6):
    return (f(x, y + h) - f(x, y - h)) / (2 * h)

# 计算在(2, 1)处的偏导数
x0, y0 = 2, 1
fx = partial_derivative_x(f, x0, y0)
fy = partial_derivative_y(f, x0, y0)

print(f"对x的偏导 f_x(2,1) = {fx:.2f}")  # 理论值=2*2*1=4，输出≈4.00
print(f"对y的偏导 f_y(2,1) = {fy:.2f}")  # 理论值=2²+9*1²=13，输出≈13.00
```


### 二、全微分：Python做“误差分析/近似计算”

全微分是“多元函数的线性近似”，对应Python中**小增量估算**，比如工程计算中快速估算参数变化带来的结果误差。

#### 应用场景

- 工程仿真中快速估算参数微小变化的影响
    
- 金融计算中估算利率、时间等变量变动对收益的影响
    

#### Python代码示例（全微分近似）

```Python
# 用全微分近似计算函数值变化
x0, y0 = 2, 1
dx, dy = 0.01, -0.02  # x,y的微小变化

# 1. 理论全微分 dz = fx*dx + fy*dy
dz = fx * dx + fy * dy
# 2. 实际函数值变化 Δz = f(x0+dx,y0+dy) - f(x0,y0)
delta_z = f(x0+dx, y0+dy) - f(x0, y0)

print(f"全微分近似 dz = {dz:.4f}")    # 输出≈4*0.01 +13*(-0.02) = -0.22
print(f"实际变化 Δz = {delta_z:.4f}") # 输出≈-0.2195（近似效果极好）
```


### 三、方向导数 & 梯度：Python实现优化算法的核心

梯度是多元微分学最有“代码价值”的概念——**所有机器学习优化算法（梯度下降、Adam等）的核心就是梯度**。

#### 应用场景

- 机器学习：梯度下降找损失函数最小值（比如线性回归、神经网络）
    
- 路径优化：找函数上升/下降最快的方向（比如机器人路径规划）
    

#### Python代码示例（梯度下降找极值）

```Python
# 用梯度下降找函数 f(x,y) = x² + 2y² 的最小值（理论最小值在(0,0)）
def f(x, y):
    return x**2 + 2 * y**2

# 计算梯度 (df/dx, df/dy)
def gradient(x, y):
    dx = 2 * x    # 对x偏导
    dy = 4 * y    # 对y偏导
    return np.array([dx, dy])

# 梯度下降核心逻辑
x, y = 3.0, 2.0  # 初始点
learning_rate = 0.1  # 学习率（步长）
epochs = 50       # 迭代次数

for i in range(epochs):
    grad = gradient(x, y)
    # 负梯度方向是下降最快的方向：更新x,y
    x = x - learning_rate * grad[0]
    y = y - learning_rate * grad[1]
    # 每10次迭代打印一次结果
    if (i+1) % 10 == 0:
        print(f"第{i+1}次迭代：x={x:.4f}, y={y:.4f}, f(x,y)={f(x,y):.4f}")

# 最终结果（趋近于(0,0)）
print(f"\n最终结果：x={x:.4f}, y={y:.4f}, 最小值={f(x,y):.4f}")
```

#### 输出结果（直观看到梯度下降的效果）

```Plain

第20次迭代：x=0.0759, y=0.0006, f(x,y)=0.0057
第30次迭代：x=0.0121, y=0.0000, f(x,y)=0.0001
第40次迭代：x=0.0019, y=0.0000, f(x,y)=0.0000
第50次迭代：x=0.0003, y=0.0000, f(x,y)=0.0000

```


### 四、多元极值 & 条件极值：Python求解优化问题

不管是无约束极值（AC-B²判断）还是条件极值（拉格朗日乘数法），都是Python解决“最优化问题”的核心方法。

#### 应用场景

- 无约束极值：比如找成本函数的最小值、利润函数的最大值
    
- 条件极值：比如“预算固定时，如何分配资源使收益最大”（拉格朗日乘数法）
    

#### Python代码示例（拉格朗日乘数法求条件极值）

```Python
from scipy.optimize import minimize  # 科学计算库，专门解优化问题

# 目标：在约束 x + y = 5 下，求 f(x,y) = x² + y² 的最小值
# 1. 定义目标函数（要最小化的函数）
def objective(vars):
    x, y = vars
    return x**2 + y**2

# 2. 定义约束条件（x + y - 5 = 0）
def constraint(vars):
    x, y = vars
    return x + y - 5

# 3. 设置约束格式（scipy要求的格式）
con = {'type': 'eq', 'fun': constraint}

# 4. 初始猜测值
x0 = [1.0, 4.0]

# 5. 调用优化函数求解
result = minimize(objective, x0, constraints=con)

# 输出结果
print("是否收敛：", result.success)
print(f"最优解：x={result.x[0]:.2f}, y={result.x[1]:.2f}")  # 理论值x=2.5,y=2.5
print(f"最小值：{objective(result.x):.2f}")                 # 理论值=12.5
```

#### 输出结果

```Plain
是否收敛： True
最优解：x=2.50, y=2.50
最小值：12.50
```
