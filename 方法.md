方法的本质是定义在类里面的函数
如果在一个方法里想调用同一个类的另一个方法，需要用 `self.方法名()` 来调用

方法和函数不同
方法：
对象名.方法名(...)
例如：list.append(...)     ->    在列表list里面添加新的内容，此时列表的内容改变
s.upper()   ->   将s（字母字符串）里面的字母全部变成大写，但是s的内容不会发生改变因为字符串不可变，如果要改变s的话应该重新进行赋值，s=s.upper()
list.remove(...)    ->    删除列表中的值，注意要删除的元素要存在于列表中否则会报错


函数：
函数名(对象)
例如：len(list)    ->    返回列表元素数量
max(list)    ->    返回列表最大值
min(list)
sorted(list)    ->    将列表从小到大排序，然后返回
